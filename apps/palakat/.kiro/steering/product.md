# Product Overview

Palakat is an event notifier mobile application built with Flutter. The app serves as a church management and communication platform, enabling users to:

- View and manage church activities (events, services, announcements)
- Browse and access a song book
- Handle activity approvals and publishing
- Manage user accounts and church memberships
- Authenticate via Firebase

The app targets Indonesian users (locale: id_ID) and integrates with Firebase for backend services.
